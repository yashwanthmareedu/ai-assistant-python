# AI Assistant using Python

This is a simple AI Assistant built using Python and open-source tools.

## Features
- Handles greetings (Hi, Hello)
- Solves basic mathematical problems
- Provides weather updates (demo)
- Generates responses using GPT-2 model

## Tech Stack
- Python
- Hugging Face Transformers
- Virtual Environment (venv)

## How to Run

1. Install dependencies:
pip install transformers torch

2. Run the program:
python main.py
